# Typeform To CSV
Simple little Typeform-to-CSV parser

## Getting Started:
It's as easy as 1, 2 ,3!

1. Clone the repo
```
$ git clone https://github/eyyu/TypeformToCSV.git
```
2. Go to Main.py and insert the typeform API key in line 25
3. Run the script!
```
$ Main.py
```
